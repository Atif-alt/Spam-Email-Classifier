\# Spam Email Classifier



A Machine Learning project that detects whether a message is Spam or Not Spam.



\## Features

\- Spam message detection

\- Text preprocessing

\- Naive Bayes classifier

\- Interactive user input

\- 97% accuracy



\## Technologies Used

\- Python

\- Pandas

\- Scikit-learn



\## Dataset

SMS Spam Collection Dataset from Kaggle



\## How to Run



py main.py



\## Example



Input:

Congratulations! You won 10000 rupees



Output:

Spam Message
